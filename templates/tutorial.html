This is the game Hangman. If you've never played the game, this is how it works:

Click New Game: the game's afoot! The chosen word is cloaked in mystery, clued only by the length of the _'s for you to follow.

You must start by guessing a letter. If it's in the word, any cases in the word where the letter is located, will reveal themselves.

If the letter is not present, a progressively desparate hanged man will appear. 7 wrong tries and he's done for.

If you guess all letters correctly, you win! And gain some bragging rights.

Protip: Start with vowelss. You'll thank me.
